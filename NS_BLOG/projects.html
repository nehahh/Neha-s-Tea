ISE ChatBot

ISE ChatBot is a web-based chat application that utilizes OpenAI's GPT-3.5-turbo model to generate responses. It serves as a conversational interface, allowing users to interact with the chatbot and receive responses.

Features:

  • User Interaction: Users can engage in conversations with the chatbot by entering messages in the input field.
  • OpenAI Integration: The chatbot leverages OpenAI's GPT-3.5-turbo model to generate responses based on user input.
  • Mental Health Awareness: The chatbot is designed to identify and respond appropriately to mental health-related queries.

Prerequisites:

  • Web browser with JavaScript enabled
  • API Key for OpenAI GPT-3.5-turbo (insert your API key in script.js)

Usage:

  • Launch the application by opening index.html.
  • Click on the chatbot icon to open the chat interface.
  • Enter messages in the chat input to engage with the chatbot.
  • The chatbot will provide responses based on the user's input.

Customization:

  • To customize the appearance, modify the styles in style.css.
  • Adjust the chatbot behavior by updating the logic in script.js.

Medical ML Project

This project uses a machine learning model to predict diseases based on user-entered symptoms. It also provides additional information such as precautions, medications, workouts, and diets for the predicted disease.

Features:

  • Predicts diseases based on user symptoms.
  • Displays disease description.
  • Provides precautions, medications, workouts, and diets for the predicted disease.

Technologies Used:

  • Python, Flask, Streamlit
  • Pandas, Scikit-learn
  • HTML/CSS (for web templates)

File Structure:

  • dataset: CSV files with data
  • models: Trained model file
  • templates: HTML files
  • app.py, train_model.py, requirements.txt, README.md

Music Data Analysis

This project involves analyzing a music dataset, including data cleaning, exploratory data analysis, clustering, and regression modeling to understand patterns and predict song popularity.

Highlights:

  • Data Cleaning: Removing duplicate entries to improve dataset accuracy.
  • Exploratory Data Analysis: Analyzing data distributions and feature relationships.
  • Clustering: Using K-means to group songs based on features.
  • Regression: Building a model to predict song popularity.
